## Description #

Plugin Check is a tool for the WordPress.org plugins review team.
This is intended as a generic tool that is not at all intended on being complete.

Setup steps:
 - `npm install`
 - `npm run wp-env start`
 - `npm run setup:tools`

Commands:
 - `npm run wp-env start`
 - `npm test`
